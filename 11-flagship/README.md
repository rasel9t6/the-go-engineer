# 11 Flagship

## Mission

This final stage is where the curriculum stops being a collection of lessons and becomes integrated
system proof.

By the end of this stage, a learner should be able to:

- navigate a larger integrated codebase
- understand how multiple engineering layers meet in one project
- use code generation as supporting leverage instead of hidden magic

## Stage Map

| Track | Surface | Core Job |
| --- | --- | --- |
| `FG.1` | [enterprise capstone](./01-enterprise-capstone) | integrated application proof surface |
| `FG.2` | [code generation](./02-code-generation) | build-time tooling and generated-code workflows |

## Suggested Learning Flow

1. Start with the [enterprise capstone](./01-enterprise-capstone).
2. Use [code generation](./02-code-generation) as a supporting late-stage toolset.

## Finish This Stage When

- you can explain how the flagship system is structured
- you can follow code across layers without losing the big picture
- you can explain where generation helps and where hand-written code should stay explicit
