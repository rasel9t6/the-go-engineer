# V2 Authoring Templates

## Purpose

This directory holds reusable v2 authoring templates derived from the approved planning and
prototype work.

These templates are maintainer-facing tooling assets on `planning/v2`.
They are not learner-facing content and should only move toward `main` when the corresponding v2
implementation work is approved.
